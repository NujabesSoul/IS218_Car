# IS218 Design a Car Factory Assignment

Download this CSV file (click to download) that contains a list of cars and produce a factory that can create an object from each car and add it to an array of car objects.   Your car needs to contain the following properties:

1. Make

2. Model

3.  Transmission

4.  Horse Power

5.  Engine Code

6.  Number of Cylinders

Each object should have a method to print the car information in HTML.  When your program runs you should be able to print a list of cars with a horizontal rule in between in car and then the information that describes the car.
